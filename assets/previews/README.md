# Preview Assets

This directory stores visual previews for the repository README and project
pages.

## ESP Web UI

- `esp-web-desktop.png` shows the setup console at a desktop viewport.
- `esp-web-mobile.png` shows the same setup console in the compact layout.

The ESP screenshots use representative mocked device data so the preview is
stable even when a microcontroller is not connected.

## Grafana

Add Grafana dashboard screenshots here with the `grafana-` prefix, for example:

- `grafana-environment-dashboard.png` shows the provisioned Atmospheric
  Environment dashboard with seeded local data.
- `grafana-anomaly-dashboard.png`
